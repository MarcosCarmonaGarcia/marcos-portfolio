# 🚀 Astro Starter Kit: Basics

Este proyecto está creado con el kit de inicio básico de Astro.

```bash
npm create astro@latest -- --template basics

/
├── public/
│   └── favicon.svg
├── src/
│   ├── assets/
│   │   └── astro.svg
│   ├── components/
│   │   └── Welcome.astro
│   ├── layouts/
│   │   └── Layout.astro
│   └── pages/
│       └── index.astro
└── package.json

| Comando                   | Descripción                                                          |
| ------------------------- | -------------------------------------------------------------------- |
| `npm install`             | Instala las dependencias                                             |
| `npm run dev`             | Inicia el servidor de desarrollo en `http://localhost:4321`          |
| `npm run build`           | Genera la versión de producción en `./dist/`                         |
| `npm run preview`         | Previsualiza la build de producción localmente                       |
| `npm run astro ...`       | Ejecuta comandos de la CLI de Astro (ej: `astro add`, `astro check`) |
| `npm run astro -- --help` | Muestra ayuda de la CLI                                              |
